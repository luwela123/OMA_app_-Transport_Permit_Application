# 🌾 OMA' (Office of Municipal Agriculturist) App

Welcome to the official repository for the OMA' Application.

## Overview

The OMA' App is designed to streamline agricultural permit applications and improve communication between farmers and the Office of the Municipal Agriculturist.

The system aims to:

- Simplify permit application processes
- Reduce paperwork and manual transactions
- Provide real-time application status tracking
- Improve data management and reporting

## System Components

### 📱 Mobile Application

The mobile application allows farmers to:

- Register and log in
- Submit permit applications
- Upload required documents
- Track application status
- Receive notifications

### 💻 Admin Dashboard

The admin dashboard allows administrators to:

- Review applications
- Approve or reject and mark paid application requests
- Generate and print permits
  message user for additional documents(through in-app notification) 
- View reports and analytics

## Technologies Used

- Flutter(dart)
- Firebase Authentication
- Firebase Realtime Database
- Node.js
- React.js
